# DESIGN.md

## 气质与意象
千禧年 Windows 2000/XP 桌面应用程序。想象2001年在图书馆机房打开一台米色CRT显示器电脑，启动一个读书协作软件的感觉。灰色窗口框架、蓝色渐变标题栏、像素化的3D按钮、Tahoma字体。

## 视觉策略
- 所有UI元素模拟经典Windows窗口控件
- 每个功能面板都是一个独立的"窗口"，带有标题栏和操作按钮
- 使用经典的 outset/inset 3D 边框模拟凹凸效果
- 图标使用16x16像素风格

## 配色方案
- 窗口背景：#C0C0C0（经典Windows灰）
- 标题栏渐变：#0A246A → #3A6EA5（深蓝到亮蓝）
- 按钮面：#D4D0C8
- 窗口边框高光：#FFFFFF / #DFDFDF
- 窗口边框阴影：#808080 / #404040
- 正文文字：#000000
- 选中高亮：#0A246A 背景 + #FFFFFF 文字
- 链接/交互：#0000FF（经典蓝色超链接）
- 批注高亮：#FFFF00（黄色荧光笔效果）

## 字体排版
- 字体族：Tahoma, 'MS Sans Serif', 'Microsoft YaHei', sans-serif
- 基础字号：11px（正文）、12px（标题）、13px（窗口标题）
- 不使用字体加粗作为强调，使用下划线或颜色变化
- 行间距紧凑，模拟经典桌面应用的密集信息排列

## 动效与交互
- 按钮点击：inset 3D 效果（边框反转），无过渡动画
- 窗口悬停：无动画，即时响应
- 批注高亮：黄色背景即时切换
- 所有交互都是"即时"的，没有 fade/slide 过渡

## 设计禁忌
- 禁止大圆角（最多1-2px）
- 禁止 box-shadow 模拟现代阴影
- 禁止毛玻璃/模糊效果
- 禁止渐变色背景（标题栏除外）
- 禁止 emoji 作为图标（使用文字或像素图标）
- 禁止 flex gap 过大的宽松布局
